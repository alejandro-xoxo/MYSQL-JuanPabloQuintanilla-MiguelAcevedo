## 📋 Descripción del Proyecto

El sistema permite gestionar de manera eficiente las operaciones básicas de una ferretería, garantizando la integridad de los datos y facilitando la consulta de información clave para la toma de decisiones. 💡

### Características principales:

* **Gestión de Inventario:** Control de existencias, mínimos y máximos. 📦
* **Administración de Productos:** Clasificación por categorías y seguimiento de estados. 🔧
* **Gestión de Proveedores:** Información detallada y estado de actividad. 🚚
* **Consultas Avanzadas:** Reportes sobre niveles de stock, proveedores activos y ofertas de suministro. 🔎

## ⚙️ Estructura del Proyecto

* `ddl.sql`: Definición de la estructura de la base de datos (tablas, relaciones y restricciones).
* `dml-data.sql`: Script con los datos iniciales para las pruebas.
* `dml-querys.sql`: Consultas SQL para la explotación de la información.
* `ERDDiagram1.svg`: Diagrama Entidad-Relación que ilustra el modelo de datos.
